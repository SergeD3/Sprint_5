# Sprint_5
# Проект автоматизации тестирования сервиса Stellar Burgers
1. Основа для написания автотестов — фреймворк pytest и selenium.
2. Так же для генерации тестовых данных используется библиотека Faker.
3. Для генерации пароля используется библиотека [password_generator](https://pypi.org/project/random-password-generator/).
4. Установить зависимости — pip install -r requirements.txt.
